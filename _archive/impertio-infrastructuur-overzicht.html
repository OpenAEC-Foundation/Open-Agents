OpenAgents — Pi.dev × ERPNext Agentic Platform
📍 github.com/Impertio-Studio/openagents · Roadmap Q1-Q2 2026
VISIE
Fase 1: Leren & Experimenteren
Pi installeren, Anthropic koppelen, basis workflows leren. Claude Code ervaring overbrengen. AGENTS.md schrijven voor ERPNext projecten.
Freek nu starten
Fase 2: OpenAgents Snippets & Extensions
Snippet bibliotheek opzetten (shared + per agent type). Smart Context extension en Model Router bouwen in TypeScript.
Freek + Pritam SDK/RPC
Fase 3: Docker Agent Deployment
Pi agents als Docker containers op Hetzner. Per klant geïsoleerd. Alleen custom API tools, geen bash/write naar host. Pritam beheert containers.
Pritam: Docker ops Freek: extensions Dockerfile in repo
Fase 4: Multi-tenant AI Platform
Pi containers per klant via Priilk Cloud. Ollama op Hetzner voor lokale LLM (classificatie/routing). White-label AI operations als dienst.
OpenCompany246 schaalbaar Ollama lokaal
